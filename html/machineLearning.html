Recursive Neural Network for image parsing

Description

Image Parsing attempts to find a semantically meaningful label for every pixel in an image. It means in one image, computer can tell this is a tree, here is the sky, that is a building and so on. Actually, we ca think image parsing problem as linguistic analysis, it also has a tree structure. Therefore, I used recursive neural network to solve this problem

My work

1. Analyzed the recursive neural network algorithm.
2. Implement it in Matlab.
